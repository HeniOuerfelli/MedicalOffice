Here you can find YourDoctor's services !




The Doctor
Create a doctor account: The practice management system allows doctors to register in the platform as a doctor, this is done by completing a registration form.

Authenticate: The practice management system provides doctors with an authentication form that they must fill out in its fields to authenticate.

Manage medical records: The practice management system allows doctors to manage medical records, including viewing a medical record and adding a medical record.

Prescribe treatment: The practice management system allows doctors to prescribe the necessary treatments to the patient.

Provide Certificate: The practice management system allows doctors to provide certificates requested by patients.

Carry out a teleconsultation: The practice management system allows doctors to carry out a teleconsultation by providing a corresponding interface.


The Secretary
Create secretary account: The practice management system allows secretaries to register in the platform as a secretary, this is done by completing a registration form.

Authenticate: The practice management system provides secretaries with an authentication form that they must fill in its fields to authenticate.

Manage patient records: The practice management system allows secretaries to manage patient records, including adding a patient record and modifying a patient record.

Manage CNAM: The practice management system allows secretaries to manage CNAM files, including adding, modifying and deleting a CNAM file.

Manage payments: The practice management system allows secretaries to manage payments, including adding a receipt, adding expenses and viewing the balance sheet.

Manage Appointments: The practice management system allows healthcare professionals to manage their appointments, including booking appointments, canceling appointments and changing appointments.


The Patient
Create patient account: The practice management system allows patients to register in the platform as a patient, this is done by completing a registration form.

Authenticate: The practice management system provides patients with an authentication form that they must fill out its fields to authenticate.

Search for a Doctor: The practice management system provides patients with a search bar containing several options; search by name, search by specialty and search by geographic location, to make it easier to find a doctor.

Make an appointment: The practice management system provides patients with an appointment form to complete in order to make an appointment.

Request a teleconsultation: The practice management system provides patients with the power to request a teleconsultation by completing a teleconsultation form.

Edit personal information: The practice management system provides patients with full access to their personal information from where they can edit it.
